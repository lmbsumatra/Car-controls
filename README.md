A Unity game project with basic car controls.

## Table of Contents

- [Controls](#controls)
- [Installation](#installation)

## Controls

- **Turning:**
  - A and D / Left and Right arrow keys: Turning left and right respectively.

- **Movement:**
  - W and S / Up and Down arrow keys: Move forward and backward respectively.

- **Gear Shifts:**
  - Pressing 1, 2, 3: Changes gear shifts.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/lmbsumatra/Car-controls.git
2. Open Unity and add the project:
   - Open Unity Hub.
   - Click on "Add" and select the cloned project folder.
3. Run the game:
   - Open the project in Unity.
   - Navigate to the main scene.
   - Press the play button to run the game.
